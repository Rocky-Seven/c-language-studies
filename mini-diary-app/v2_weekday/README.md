# v2_weekday

対応記事: [C言語で ミニ日記アプリを作ろう！（2）](https://my-studies.org/lets-make-a-mini-diary-app-using-c-02/)

## 概要

Ver.1 に、日時表示への「曜日（日本語）」追加を行ったバージョン。

### 追加した機能

- 曜日を表示（日本語対応）：`weekdays_jp[]` 配列 + `snprintf()` で組み立て
- ロケールを `ja_JP.UTF-8` に設定し、MSYS2/UCRT64環境での日本語表示を安定化

### Ver.1（`strftime`）との違い

| 項目      | Ver.1: `strftime` | Ver.2: `snprintf` |
| ------- | ----------------- | ------------------ |
| 曜日表示    | ❌ 不可（別途処理が必要）      | ✅ 日本語で可能            |
| カスタマイズ性 | 制限あり               | 自由度が高い              |
| 実装の手軽さ  | 簡潔                 | やや複雑                |

`timestamp` バッファも、曜日や装飾を含められるよう `char[100]` → `char[128]` に拡張している。

## コンパイル・実行

```
gcc -o diary_v2 diary_v2.c
./diary_v2
```

## 実行例

```
今日の日記を入力してください（Enterで終了）：
今日はお天気になりました。
日記を保存しました。

$ cat diary.txt
[2025-08-08（金） 10:11:53]
今日はお天気になりました。
```
