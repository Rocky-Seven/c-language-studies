# 03: 選択ソート（C言語版）

対応記事: 「【基本情報技術者試験】選択ソート問題をC言語のコードで確認してみよう！」

## 概要
- `selection_sort`：未整列部分から最小値を探し、先頭の要素と交換することを繰り返す探索
- 1パスにつき交換は最大1回だけなので、比較回数は多いが交換回数は少ないのが特徴

パスごとの配列の状態をその場で表示するようにしているので、コードを読むだけでなく
実行結果からも「外側ループが1回進むたびに、どこまで確定していくか」を体感できるようにしています。

## 実行方法
```bash
gcc -Wall -Wextra -o selection_sort selection_sort.c
./selection_sort
```

## 実行結果例
```
ソート前: 5 2 8 1 9 3
1回目のパス後: 1 2 8 5 9 3
2回目のパス後: 1 2 8 5 9 3
3回目のパス後: 1 2 3 5 9 8
4回目のパス後: 1 2 3 5 9 8
5回目のパス後: 1 2 3 5 8 9
ソート後: 1 2 3 5 8 9
```

## 計算量
- 時間計算量: O(n²)（データの並び方に関係なく、最良・最悪・平均すべて同じ）
- 交換回数: 最大n-1回（バブルソートより少ない）
- 安定性: 不安定（同じ値の要素の順序が入れ替わる場合がある）
