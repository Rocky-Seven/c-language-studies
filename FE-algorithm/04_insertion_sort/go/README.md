# Go版
C言語版の挿入ソートをGoに移植したものです。
## 実行方法（Codespaces等、Go導入済み環境で）
```bash
go run insertion_sort.go
```
## 実行結果例
```
ソート前: [5 2 8 1 9 3]
1回目のパス後: [2 5 8 1 9 3]
2回目のパス後: [2 5 8 1 9 3]
3回目のパス後: [1 2 5 8 9 3]
4回目のパス後: [1 2 5 8 9 3]
5回目のパス後: [1 2 3 5 8 9]
ソート後: [1 2 3 5 8 9]
```
## C言語版との違い
- **ループの表現**：Cの `while` 文はGoには存在せず、条件式だけを書いた `for` 文で同じ動きを表現します
- **配列の長さ**：Cでは `#define SIZE` で自分で管理していましたが、Goは `len(arr)` でいつでも取得できます
- **表示処理**：`fmt.Println` は配列をそのまま渡すだけで中身を表示できるので、Cの `printf` でループを回して1要素ずつ出力するより手軽に書けます
