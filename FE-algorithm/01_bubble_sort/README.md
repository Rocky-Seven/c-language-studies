# 01: バブルソート

基本情報技術者試験のアルゴリズム問題（バブルソート）を、まずC言語で実装し、
今後Go / Kotlinへの移植を通じて「言語ごとの書き方の違い」を比較していく企画です。

## 対応記事

- [基本情報技術者試験のバブルソート問題をC言語のコードで確認してみよう](https://my-studies.org/)（公開後にURLを更新）
- （準備中）Go/Kotlin移植編：C版で書いたバブルソートを移植し、言語ごとの違いを比較

## 移植のポイント（執筆時にチェックする観点）

- 値の交換処理の書き方の違い（一時変数が必要か、糖衣構文があるか）
- 配列とスライス／リストの扱いの違い
- ループ構文の書き方の違い（`for`の範囲指定、`range`の有無など）
- 型の扱い（静的型付け同士でも、型推論の強さに違いがあるか）

各言語のディレクトリの実装が完了したら、この観点でまとめ記事を書く想定です。
