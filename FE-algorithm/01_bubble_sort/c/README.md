# 01: バブルソート

対応記事: 「基本情報技術者試験のバブルソート問題をC言語のコードで確認してみよう」
関連記事: [C言語のきほん「バブルソート」](https://my-studies.org/c-kihon-bubble-sort/)（実装の基礎編）

## 概要

隣り合う要素を比較・交換しながら整列させる、最も基本的なソートアルゴリズムの実装です。
`swapped` フラグによる早期終了処理を入れています。

## 実行方法

```bash
gcc -Wall -Wextra -o bubble_sort bubble_sort.c
./bubble_sort
```

## 実行結果例

```
ソート前: 5 2 8 1 9 3
1回目のパス後: 2 5 1 8 3 9
2回目のパス後: 2 1 5 3 8 9
3回目のパス後: 1 2 3 5 8 9
4回目のパス後: 1 2 3 5 8 9
5回目のパス後: 1 2 3 5 8 9
ソート後: 1 2 3 5 8 9
```

## 計算量

- 最悪計算量: O(n²)
- 安定ソート（同じ値の相対順序が保たれる）
